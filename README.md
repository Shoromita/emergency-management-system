A GUI-based Emergency Management System built with Python and Tkinter for coordinating emergency response.

## Features

- **User Registration & Login** - Report and track emergencies
- **Volunteer Management** - Automatic assignment to emergencies
- **Admin Dashboard** - Complete system control
- **Resource Tracking** - Manage emergency resources
- **CSV Data Storage** - Persistent data storage

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/emergency-management-system.git
   cd emergency-management-system
   ```

2. **Install dependencies:**
   ```bash
   pip install pandas
   ```

3. **Run the application:**
   ```bash
   python emergency.py
   ```
